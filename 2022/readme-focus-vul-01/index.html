CVE-2022-22954

VMware Workspace ONE Access and Identity Manager 包含一个由于服务器端模板注入而导致的远程代码执行漏洞。具有网络访问权限的恶意行为者可以触发可能导致远程代码执行的服务器端模板注入。

  • https://github.com/sherlocksecurity/VMware-CVE-2022-22954

  • https://github.com/aniqfakhrul/CVE-2022-22954

  • https://github.com/Vulnmachines/VMWare_CVE-2022-22954

  • https://github.com/jax7sec/CVE-2022-22954

  • https://github.com/lucksec/VMware-CVE-2022-22954

  • https://github.com/axingde/CVE-2022-22954-POC

  • https://github.com/chaosec2021/CVE-2022-22954-VMware-RCE

  • https://github.com/mumu2020629/-CVE-2022-22954-scanner

CVE-2022-26258

D-Link DIR-820L 1.05B03 通过 /lan.asp 中的设备名称参数被发现包含远程命令执行 (RCE) 漏洞。

  • https://nvd.nist.gov/vuln/detail/CVE-2022-26258

CVE-2022-25106

通过函数 genacgi_main 发现 D-Link DIR-859 v1.05 包含基于堆栈的缓冲区溢出。此漏洞允许攻击者通过精心设计的有效负载引发拒绝服务 (DoS)。

  • https://nvd.nist.gov/vuln/detail/CVE-2022-25106

CVE-2022-22908

深信服 VDI 客户端 5.4.2.1006 中的 SangforCSClient.exe 允许攻击者在能够读取进程内存时发现用户名和密码字段的内容。

  • https://nvd.nist.gov/vuln/detail/CVE-2022-22908

CVE-2022-20699

Cisco Small Business RV160、RV260、RV340 和 RV345 系列路由器中的多个漏洞可能允许攻击者执行以下任何操作： 执行任意代码 提升权限 执行任意命令 绕过身份验证和授权保护 获取并运行未签名的软件 导致拒绝服务 ( DoS）有关这些漏洞的更多信息，请参阅本公告的详细信息部分。

  • https://github.com/rohankumardubey/CVE-2022-20699

  • https://github.com/Audiobahn/CVE-2022-20699

CVE-2022-20660

多个 Cisco IP 电话型号的信息存储架构中的漏洞可能允许未经身份验证的物理攻击者从受影响的设备获取机密信息。此漏洞是由于受影响设备上的机密信息未加密存储造成的。攻击者可以通过物理提取和访问其中一个闪存芯片来利用此漏洞。成功的利用可能允许攻击者从设备获取机密信息，这些信息可用于后续攻击。

  • https://nvd.nist.gov/vuln/detail/CVE-2022-20660
